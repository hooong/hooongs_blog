# [백준9372번] 상근이의 여행 / Python3

## 문제

상근이는 겨울방학을 맞아 N개국을 여행하면서 자아를 찾기로 마음먹었다. 

하지만 상근이는 새로운 비행기를 무서워하기 때문에, 최대한 적은 종류의 비행기를 타고 국가들을 이동하려고 한다.

이번 방학 동안의 비행 스케줄이 주어졌을 때, 상근이가 **가장 적은 종류**의 비행기를 타고 모든 국가들을 여행할 수 있도록 도와주자.

상근이가 한 국가에서 다른 국가로 이동할 때 다른 국가를 거쳐 가도(심지어 이미 방문한 국가라도) 된다.

## 입력

첫 번째 줄에는 테스트 케이스의 수 T(T ≤ 100)가 주어지고,

각 테스트 케이스마다 다음과 같은 정보가 주어진다.

- 첫 번째 줄에는 국가의 수 N(2 ≤ N ≤ 1 000)과 비행기의 종류 M(1 ≤ M ≤ 10 000) 가 주어진다.
- 이후 M개의 줄에 a와 b 쌍들이 입력된다. a와 b를 왕복하는 비행기가 있다는 것을 의미한다. (1 ≤ a, b ≤ n; a ≠ b) 
- 주어지는 비행 스케줄은 항상 연결 그래프를 이룬다.

## 출력

테스트 케이스마다 한 줄을 출력한다.

- 상근이가 모든 국가를 여행하기 위해 타야 하는 비행기 종류의 최소 개수를 출력한다.

## 예제 입력 1

```
2
3 3
1 2
2 3
1 3
5 4
2 1
2 3
4 3
4 5
```

## 예제 출력 1

```
2
4
```

## 나의 풀이

> 해당 문제는 최소 신장 트리를 이해하는지를 물어보는 문제였습니다. 거창한 입력이 주어지는 것 같지만 다소 허무한 답을 요구하는 문제여서 살짝 당황을 했던 문제입니다..ㅎㅎ 
>
> 풀이는 주어지는 N개의 노드를 잇는 최소 신장 트리를 구하는 것이었지만 여기서 간선의 가중치는 모두 같다고 보기때문에 사이클이 없는 그래프를 구하면 되는 문제입니다. 그런데 문제에서 주어지는 입력이 항상 연결그래프라고 나오기때문에 모든 노드가 다른 노드와 연결되는 간선을 하나씩만 가지기때문에 그 간선의 개수만 구해주면 되는겁니다. 즉, `N-1`이 정답입니다..
